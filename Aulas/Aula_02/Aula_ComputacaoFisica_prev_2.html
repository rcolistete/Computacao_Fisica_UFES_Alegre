Introduo ao uso de BBC micro:bit com MicroPython

Todo o cdigo-fonte listado a seguir MicroPython e deve ser rodado no microcontrolador BBC micro:bit.

Software Mu Editor conectado via cabo microUSB a microcontrolador BBC micro:bit rodando MicroPython

Ao iniciar o Mu Editor, clique no boto "Modo" ( o 1o esquerda, tem o cone da cobrinha de Python) para confirmar que est no modo "BBC micro:bit" ( o 1o na lista vertical de modos).

Conecte o cabo microUSB entre a porta USB do computador e a porta microUSB do BBC micro:bit :

  • um LED amarelo deve acender ao lado do conector microUSB, e no BBC micro:bit v2 tambm acende um LED vermelho do outro lado do conector microUSB;
  • deve aparecer uma mensagem no sistema operacional (Linux, Mac OS ou Windows) para montar o dispositivo "MICROBIT", como se fosse um pendrive, confirme.

Caso no tenha MicroPython instalado no BBC micro:bit, ento crie um novo documento MicroPython vazio clicando no boto "Novo" ( o 2o esquerda, tem o cone de "+"), depois clique no boto "Transferir" ( o 5o esquerda, tem o cone de uma pequena seta para baixo apontando para o cone de BBC micro:bit).

Com MicroPython instalado no BBC micro:bit, clique no boto "REPL" ( o 7o contando d esquerda, tem o cone de teclado), ento uma janela inferior ser exibida no Mu Editor com ttulo "REPL BBC microbit".

"REPL" quer dizer "Read, Evaluate, Print, and Loop", ou seja, o modo interativo tanto em Python como em MicroPython.

Mensagem ao iniciar o modo interativo/terminal do MicroPython em BBC micro:bit v1.x :

MicroPython v1.9.2-34-gd64154c73 on 2017-09-01; micro:bit v1.1.1 with nRF51822
Type "help()" for more information.
>>>

E em BBC micro:bit v2 :

MicroPython v1.15-64-g1e2f0d280 on 2021-06-30; micro:bit v2.0.0 with nRF52833
Type "help()" for more information.
>>>

Uso interativo de MicroPython no Mu Editor

Na janela interativa (REPL) do Mu Editor, digite "21+21" e a tecla Enter, o texto de entrada aparece aps o cursor ">>>" e o resultado (42, citao do livro "O Guia do Mochileiro das Galxias", de Douglas Adams) na linha seguinte (mas sem as cores de sintaxe na janela REPL) :

>>> 21+21
42

Pode-se usar na janela interativa (REPL) :

  • setas para cima e para baixo para acessar as ltimas linhas digitadas;
  • tecla TAB para completar nomes de comandos MicroPython, variveis, funes, etc;
  • teclas HOME (ou Ctrl+A) e END (ou Ctrl+E) para ir para incio ou final da linha;
  • boto esquerdo do mouse para selecionar texto em uma ou mais linhas;
  • teclas Shift+seta esquerda e Shift+seta direita para selecionar texto na mesma linha;
  • teclas Ctrl+Shift+C ou boto direito do mouse para copiar texto j selecionado;
  • teclas Ctrl+Shift+V ou boto direito do mouse para colar texto;
  • teclas Ctrl+C para interromper um programa MicroPython rodando;
  • teclas Ctrl+D para dar reinicializar (soft reset) o MicroPython, apagando toda a memria de linhas digitadas, variveis definidas, funes definidas, etc, no modo interativo;
  • teclas Ctrl+E para entrar no modo "colar", permitindo colar mais de uma linha de texto, til para copiar blocos de cdigo-fonte MicroPython;
  • mouse para mudar o tamanho da janela ao clicar e arrastar na parte superior;
  • os botes "Aumentar" (cone de lupa com sinal de "+") e "Diminuir" (cone de lupa com sinal de "-") para alterar o tamanho dos caracteres de texto (altera tambm a janela superior de documentos).

gc : gerenciando memria RAM livre em MicroPython

A memria RAM livre do BBC micro:bit com MicroPython instalado pequena, ainda mais na v1.x : 8-9 KB ! Mas no BBC micro:bit v2 tem bem mais, uns 62 KB livres. Basta usar o mdulo MicroPython gc (de "garbage collector") e a funo gc.mem_free() para obter o nmero de bytes livres da RAM :

>>> import gc
>>> gc.mem_free()
63984

E tal memria RAM vai sendo consumida ao usar o MicroPython no modo interativo, rodar programas, etc. Pode-se reorganiz-la e optimiz-la, liberando mais memria livre, via gc.collect()

>>> gc.collect()
>>> gc.mem_free()
64032

Leitura de botes A e B do BBC micro:bit

O BBC micro:bit tem 2 botes na parte frontal (ao lado da tela de 5x5 LED's), nomeados boto A e boto B e funcionam como entrada de dados interativa para um usurio interagir.

Referncias :

  • Buttons tutorial;
  • micro:bit Micropython API - Buttons;
  • Microbit Module Buttons.

O mdulo MicroPython microbit precisa ser importado para a leitura dos botes. Exemplo interativo, na 2a vez o boto A estava pressionado :

>>> from microbit import *
>>> button_a.
get_presses     is_pressed      was_pressed
>>> button_a.is_pressed()
False
>>> button_a.is_pressed()
True

Aqui um loop infinito (pode ser cancelado com Ctrl+C) lendo boto A para aumentar um contador e mostrar na janela interativa (REPL) o nmero de A's pressionados via print() do (Micro)Python :

from microbit import *
n = 0
while True:
    if button_a.is_pressed():
        n += 1
    print("A"*n)
    sleep(200)

Uso de tela de 5x5 LED's vermelhos do BBC micro:bit

O BBC micro:bit tem uma matriz quadrada de 5x5=25 LED's (Light Emitting Diode - Diodo Emissor de Luz) vermelhos na parte frontal, e funcionam como sada de dados interativa para um usurio visualizar caracteres fixos, texto rolando, imagens, animaes, grficos, etc.

Referncias :

  • Hello World tutorial;
  • Images tutorial;
  • micro:bit Micropython API - Display;
  • Microbit Module - Display;
  • Microbit Module - Image.

O submdulo display do mdulo MicroPython microbit precisa ser importado para usar a tela de 5x5 LED's.

Leitura de sensor de temperatura do BBC micro:bit

O BBC micro:bit tem um sensor de temperatura no microcontrolador, que usualmente mais quente que a temperatura ambiente.

Referncias :

  • micro:bit Micropython API - the microbit module;
  • Microbit Module - Temperature.

O mdulo MicroPython microbit precisa ser importado para a leitura da temperatura do BBC micro:bit, a funo temperature() retorna um valor inteiro em graus Celsius :

>>> from microbit import *
>>> temperature()
27

Leitura do sensor de luz via os 5x5 LED's do BBC micro:bit

O BBC micro:bit consegue usar os 5x5 LED's de forma reversa (LED diodo emissor de luz) como sensor de luz aproximado, retornando um nmero entre 0 (zero, escuro) e 255 (brilho mximo de luz), mas sem calibrao com alguma unidade fsica (lux, etc).

Referncias :

  • micro:bit Micropython API - the microbit module;
  • Microbit Module - Display.

O mdulo MicroPython microbit precisa ser importado para a leitura da intensidade luminosa sobre a parte frontal do BBC micro:bit, via funo read_light_level() :

>>> from microbit import *
>>> display.read_light_level()
174
